# timetracking

Time-Tracking App created based on Electron and VueJS
## Project setup
```
npm install
```
### Run Electron Mode
```
npm run electron:serve
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
